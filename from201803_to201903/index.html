1年次の生活をざっくりと振り返る(2018年3月〜2019年3月)

  2019/03/31

後でになって忘れてしまうのはさみしいから書き残しておきます。

入学まで

いきなり重いですが、２月あたりに病んでいたんですよね。 受験勉強が中途半端で終わりになってしまう感覚があったり、時間の過ごし方や勉強の方法が下手だったことを思い返したり。 高校の授業が終わり人と会う機会が減ったのもあり、落ち込んでいて全然勉強していませんでいた。 逆にそれまでの一年間はしっかり勉強してたのはえらい。

そんな状態で電通大受験を迎えたのもあって、二次試験の当日、本来必要だったセンター試験の受験票を忘れたんですよね。 手続きをしたらセーフだったので良かったです。

受験時の席が大学A棟の一階の入り口すぐの椅子だったのですが、流れ込む冷気がひどく寒かったのを覚えています。 生協で温かいコーンスープと温かいミルクティーを買いました。

試験問題への感想は多分ほとんどの人と同じです。帰りの電車で \pi / 2 = x + \cos x \Leftrightarrow x = \pi / 2であることに気づいたり、 ツイッターで「電通大」で調べたりをしていました。このころはツイッターも検索用ツールでした。

そんな精神状態なので合否を見るのが怖くて、発表前日はずっと現実逃避でネットとかをしていました。 発表当日は起きたのが10時半過ぎ頃で、でも受かっていてよかったです。

3月

精神は荒んでいましたが合格です。さっき書いた通りです。

正しい生活をせねばという気持ちが強く、「漫画でわかる続ける技術」なる本を購入しました。読みやすくためにもなる内容で良かったです。

ポケットモンスターウルトラムーンも買います。気の迷いで言語を英語にして始めてしまい(後から変更できない)、 すぐしんどくなったのは一体何がしたかったのか意味がわかりませんが、なんとか殿堂入りまではできました。

だから今も御三家ポケモンの日本語名すら覚えていないです。

それから、近所の個人指導の塾のバイトを始めます。はじめてのアルバイトです。

中学生を中心に個別で教えるタイプなのでかなり楽な部類だったと思います。

塾の経営者さんが電通大のことを知らず、塾なのにそれでいいのかとは思いました。

薄給のうえ全然入れなかったからほとんど稼ぎはありませんでした。

趣味について、ベアタンクにしていた水槽を再開させました。時間が余っていたからめっちゃくちゃレイアウトを考えていました。

他には、Javaを勉強し直していました。「スッキリわかるJava入門 実践編」を読み返したりとか。

高2ちょっと前からプログラミングに1年間触れて、それで情報系、電通大を選んだ自分は正しかったと思ってます。

アイマスもちょっと再開しましたが、ゲームのアイマスをしたい気持ちが少なく、ログインくらいしかしていなかったはずです。

高2の初め頃デレステを始め、他のアイマスにものめり込み、学園祭のオタクの雰囲気に魅了されて電通大を選んだ当時の自分は正しかった…(嘘です)

4月

サークル見て回ったり、初対面の人とも緊張しつつも交流したり、、、はみなさんと同じはずですね。

サークルとかは何も考えてなかったのですが、色々見てみてIMLとテニス部に入りました。

活動内容がはっきりしていてしっかりしてそうなのを理由にIMLに入り、中学のときにしていたテニスを体験入部でしてみたらやはり楽しかったのでテニス部です。

どちらも楽しかったのですが、自分にはハードだったのもあり5月にIML、6月にはテニス部をどちらも辞めてしまいました。

多分、この4月が今までの人生で一番頑張っていました。新しい環境に体力を使うタイプなのかもしれません。大変でしたが充実していたと思います。

5月

少しずつ慣れてきたのかもしれません。

が、重い重い一年次実験のレポートがのしかかってきます。たぶん多くの人と同じ状況です。

部の先輩がプログラミングのアドバイスをしてくれたり、同学年にもとてもパソコンに強い人がいたりなど、良い経験でした。 部活も居心地良く、体を動かす楽しさも久々に取り戻したりしていました。

他には、そうですね、この頃からツイッターで1日数ツイートをし始めます。今もですが。

6月

中間テストです。この頃は電車内で課題を進めたり、電車で立ちながら寝たりしてました。人間立ちながら寝れるんですね。

一人暮らしをしたい気持ちが芽生え始めましたが、どうしたって無理だよなぁとも思っていました。

また、部活辞めてフリーになり、なんだか虚ろでした。

生活にゆとりは生まれましたがなんだか心が空白です。放課後は同じクラスの人と課題を進めたりをしていました。

7月

ほんの少し空いた時間に、バイトを入れ始めました。頭の良い生徒もそうでない生徒も、どちらも教えるのに楽しさも難しさもあります。

自分も問題を解けないときは焦ります。これは講師の側に問題がありますね。

大学の方では、その後の進路を分ける類分けがあるので勉強せねばの気持ちが湧いてきます。

忙しさは水槽にも現れます。良い状態をキープしておくことが無理になり、水槽がコケ始めます。

8月

期末試験を乗り越え、夏休みです。

個別指導バイトは入れる時間が少なかったのもあり、単発のバイトを入れるサービスに登録します。 入会の説明を聞くためだけに東京に行かされて、安くない交通費がかかるの本当に意味が無かったです。

実際に工場で軽作業をするのですが、その軽作業がこれまた本当に嫌で嫌で、２日でおしまいにしました。

１日中単純作業をするのは精神がやられます。感じ悪い人もいて最悪でした。普通の人も全然いましたが。

空いている時間はコンピューターの授業の復習とかをしていました。

Linuxやネットワークの仕組みまるで知らなかったので、基礎を学べて良かったです。

授業でRubyに触れました。動的型付けや文法の柔らかさに戸惑っていたのが嘘のように今はRubyが大好きです。

その劣悪工場バイトのあと、狂ったようにパソコンのバイト探したんですよね。

３日間位ずっとインターネットでネット検索したり検索したり検索したりをしていました。

大学の授業時間に合う条件と、求めるスキルが自分でもOKなところを探し出して、３つくらい申し込みました。

1つ承諾してくださった会社があり、面接をしました。有り難いことに採用していただき、それが８月は終わりです。

9月

アルバイトは研修から始まったのですが、その間もお金はいただけるので感覚がおかしくなります。

早く技術を身に着けねばといった緊張感がとてもあり、大変でしたが充実していた時期、の第二弾です。

話は飛んで、６月頃から電通大生さんのブログ記事集めまくって読みまくっていたのですが、読むと技術系バイトについての記事が割とあり、 それに触発されて技術系アルバイトを探したというのもあります。

個別指導のアルバイトはその流れで自然消滅でした。担当の生徒がどうだこうだ、みたいないざこざが無かったのが幸いです。

10月

大学が始まるのですが、授業を疎かにし始めます。コンピューター系の基礎知識を早く覚えないとという気持ちと、 バイトで体がひどく疲れる(バイト自体はとても楽しかったです)のがあり、 その気持ちはこの時期から数ヶ月ずっと続くのですが、 それが原因で授業をサボり始めます。で、これはもうずっと続きます。

11月

10月と同じです。この頃から情報系の知識を身につける楽しさをより強く感じるようになってきた気がします。

12月

11月と同じです。UECアドベントカレンダーはたのしかったです。

1月

大学へ向かう気力が回復することはなく、精神不安定になり始めます。またですか。

2月

大学を面倒に思う気持ちがより高まり、より精神不安定になります。

ですが期末試験が過ぎると体調は回復し、そこそこ健康になりました。

2月の後半は長野まで、信州大へ行った高校の友人に会いに行きました。

3月

健康です。健康診断行ってないんですよね。行かないとです。
